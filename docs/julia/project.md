# Project

## Create Project

Go to Julia console

```sh
julia
```

Go to the package mode

```sh
julia> ]
```

Generate a project

```sh
pkg> generate MyProject
```

Enter shell mode

```sh
pkg> ;
```

Got to the project folder and create remaining folders

```sh
shell>cd MyProject 
mkdir docs test
```
