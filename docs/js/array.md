# Arrays

## Push

Push arguments:

```js
let a = [1,2];
a.push(3,4)
console.log(a)
```
```output
[ 1, 2, 3, 4 ]
```

Push an array:

```js
let a = [1,2];
a.push(...[3,4])
console.log(a)
```
```output
[ 1, 2, 3, 4 ]
```
