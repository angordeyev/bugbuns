# Headers Naming

Headers of all levels are named as headers using words starting with uppercase letter,
6 levels maximum.

    # Header Level 1
    ## Header Level 2
    ### Header Level 3
    #### Header Level 4
    ##### Header Level 5
    ###### Header Level 6

# Header Level 1
## Header Level 2
### Header Level 3
#### Header Level 4
##### Header Level 5
###### Header Level 6
