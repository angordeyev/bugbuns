## Reference Font

Add to `app.css`

    @font-face {
      font-family: "Robot-Regular";
      src: url(/fonts/Roboto-Regular.ttf);
      font-weight: normal;
      font-style: normal;
    }
    .local-font {
      font-family: "Roboto-Regular";
      font-style: normal;
      font-size: 16px;
      color: red;
    }
