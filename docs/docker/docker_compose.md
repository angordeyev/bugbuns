# Docker Compose

## Commands

### Run

Run

    docker compose up

Take changes, run, and build

    docker compose up --build

Run detached

    docker compose up -d

### Stop

    docker compose down

### Logs

    docker compose logs

## Configuration File

The default file is `docker-compose.yml`

https://docs.docker.com/compose/compose-file/#version-top-level-element
